# 🌊 Aqua Dive Egypt

A responsive diving website built with HTML, Tailwind CSS, daisyUI, and Vite. The website showcases diving courses, dive sites, an image gallery, and contact information with a clean and modern user interface.

## 🚀 Features

- Responsive navigation bar
- Light / Dark theme toggle
- Hero section with image carousel
- Diving courses
- Dive sites
- Gallery
- Contact section
- Responsive footer
- Modern UI using daisyUI components

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- daisyUI
- Vite

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/jillan-hussein/blue-ocean.git
```

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Build the project:

```bash
pnpm build
```
