# 🌊 Aqua Dive Egypt

A responsive diving website built with HTML, Tailwind CSS, and daisyUI.

## 🚀 Features

- Responsive navigation bar
- Dark / Light theme toggle using daisyUI
- Diving courses section
- Dive sites menu
- Gallery section
- Contact section
- Modern UI design

## 🛠️ Technologies

- HTML5
- Tailwind CSS
- daisyUI
- Vite
